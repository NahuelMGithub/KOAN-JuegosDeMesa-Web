nombre del proyecto: KOAN

Descripción: Es una web de mi emprendimiento de Juegos de Mesa. Con unos socios tenemos una pequeña editorial de juegos inventados
             y producidos por nosotros


Link para ver el modelo en figma: https://www.figma.com/file/62q1gh0JHCjDmNayajLPYB/Untitled?type=design&node-id=0%3A1&mode=design&t=W1pgHNopKSh5XVLG-1
Se encuentra público y es el que entregue. De todas formas, algunas cosas van modificadas, porque se me ocurrieron mientras hacía el curso.



Explicación para tutor:

las secciones trabajadas son
Index, Juegos y Nosotros

Cosas hechas con el objetivo de aprender:
Importé 2 tipografías de google fonts. una para titulos y otra para contenido general. 
Use boostrap para header y footer y carrusel Ver: "Juegos en venta", "Prototipos" y Tiendas adheridas"
Use grid para ubicar imagen y texto. Ver "Nosotros".
